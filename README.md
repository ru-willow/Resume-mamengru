## GitHub Pages 设置

上传到 GitHub 后，在仓库中进入：

```text
Settings -> Pages -> Build and deployment
```

然后选择：

```text
Source: Deploy from a branch
Branch: main
Folder: /docs
```

保存后，GitHub 会生成一个公开访问地址，形式通常是：

```text
https://你的用户名.github.io/仓库名/
```

正式简历页面由 `docs/index.html` 提供展示。
